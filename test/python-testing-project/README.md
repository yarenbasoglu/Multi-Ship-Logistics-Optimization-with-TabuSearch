# Python Testing Project

## Overview
This project is designed to demonstrate a simple Python application with a clear structure, including main functionality and utility functions. It also includes unit tests to ensure the reliability of the code.

## Installation
To set up the project, follow these steps:

1. Clone the repository:
   ```
   git clone <repository-url>
   ```
2. Navigate to the project directory:
   ```
   cd python-testing-project
   ```
3. Install the required dependencies:
   ```
   pip install -r requirements.txt
   ```

## Usage
To run the main application, execute the following command:
```
python src/main.py
```

## Testing
To run the tests, you can use the following command:
```
pytest tests/
```
or
```
python -m unittest discover -s tests
```
